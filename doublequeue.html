Double Queue Implementation

Algorithm of Double Queue

1. Initialize an empty queue.
2. EnqueueFront(value):
   - Insert value at the front of the queue.
3. EnqueueRear(value):
   - Insert value at the rear of the queue.
4. DequeueFront():
   - Remove value from the front if the queue is not empty.
5. DequeueRear():
   - Remove value from the rear if the queue is not empty.
6. Display the queue contents.
